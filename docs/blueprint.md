# **App Name**: Sanvedana Admin

## Core Features:

- Dashboard Overview: Displays key metrics and recent activities at a glance, including social media reach.
- Student Management: Create, read, update, and delete student records, including personal information, medical notes, and status, with social sharing options.
- Staff Management: Manage staff profiles, roles, departments, and contact information.
- Donation Tracking: Record and track donations, manage donor information, and generate reports, with social media thank you options.
- Social Media Management: Create, schedule, and analyze social media posts across multiple platforms.
- Settings Configuration: Configure organization profile, account settings, notification preferences, and social media auto-posting preferences.
- AI-Powered Insights Tool: Generate insights and predictions on student progress or fund allocation.

## Style Guidelines:

- Primary color: Blue (#3B82F6), conveying trust and stability.
- Background color: Light gray (#F9FAFB), providing a clean and neutral canvas.
- Accent color: Soft lavender (#B19CD9), for interactive elements adding a touch of sophistication.
- Facebook: #1877F2, for clear Facebook integration.
- Instagram: Linear gradient (#F58529 to #DD2A7B to #8134AF), for clear Instagram integration.
- Headline font: 'Space Grotesk' sans-serif for headlines, reflecting modern, tech-inspired design.
- Body font: 'Inter' sans-serif for body text, ensuring optimal readability.
- Lucide React icons provide a consistent, modern look throughout the dashboard.
- Responsive grid layout for optimal viewing on mobile, tablet, and desktop devices.
- Subtle transitions and hover effects for interactive elements, enhancing user experience.
